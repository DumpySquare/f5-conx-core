Class F5Client

Main F5 connectivity client

Basic Example:

const mgmtClient = new f5Client(
     host: '192.0.2.1',
     user: 'admin',
     password: 'admin',
     {
         port: 8443,
         provider: 'tmos'
     }
);
await f5Client.discover();
const resp = await f5Client.makeRequest('/mgmt/tm/sys/version');
